## Опис проекту
Цей проект реалізує відтворюваний MLOps-цикл для класифікації текстових даних із набору **20 Newsgroups**, порівнюючи ефективність основної моделі **Multinomial Naive Bayes** з альтернативами (**Logistic Regression**, **Linear SVC**). Система інтегрує **DVC** для версіонування великих даних і моделей, **MLflow** для моніторингу параметрів експериментів та **scikit-learn Pipeline** для автоматизації обробки тексту, що забезпечує повну прозорість розробки та легке розгортання в середовищі веб-додатків.

## Troubleshooting
### Робочі директорії
Дуже важливо виконувати команди з правильних папок, щоб уникнути помилок шляхів:
* **`git` команди:** запускайте з кореневої папки всього репозиторію (`MLOps`).
* **`dvc` команди (push/pull):** запускайте з папки конкретної лабораторної (`lab1`).
* **`python` скрипти:** запускайте з папки `lab1` (наприклад, `python src/models/train.py`). Це автоматично вирішує проблему з `ModuleNotFoundError`.

### Часті помилки
* **`dvc push` fails (помилка відправки даних):** Перевірте, чи існує папка віддаленого сховища (наприклад, `/tmp/dvc-storage`). Якщо її немає, створіть її вручну: `mkdir -p /tmp/dvc-storage`.
* **`mlflow ui` не відкривається або зависає:** Швидше за все, стандартний порт 5000 вже зайнятий іншим процесом. Запустіть інтерфейс на іншому порту командою: `mlflow ui --port 5001`.
* **`ModuleNotFoundError` під час імпорту модулів:** Переконайтеся, що ви знаходитесь у правильній директорії (`lab1`), а також перевірте, чи активоване ваше віртуальне середовище (`venv`).
